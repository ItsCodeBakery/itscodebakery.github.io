XDR-NET: Hybrid CNN–Transformer Network for Diabetic Retinopathy Detection

Syed Shayan Ali Shah, et al. | Manuscript in Preparation

Project Overview

XDR-Net is an explainable hybrid CNN–Vision Transformer (ViT) model designed for automated Diabetic Retinopathy (ICDR 0–4) grading from high-resolution fundus images. The architecture effectively combines local feature extraction (CNN) with global spatial reasoning (ViT) to accurately detect multi-scale lesions (microaneurysms, hemorrhages, exudates) under varying image conditions.

The model achieved superior performance on the APTOS-2019 dataset, boasting 97.30% Validation F1-score and 97.88% AUC. It also provides calibrated probabilities and Grad-CAM heatmaps for case-level interpretability.
